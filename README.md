# Телеграмм-бот(игра) "Покемон"

---

## Особенности

- **Рандомная генерация:** Каждый новый покемон получает случайного персонажа с картинкой, уникальными характеристиками (HP, атака, защита).
- **Классы персонажей:** Выбери класс для своего покемона!
    - **Обычный:** Сбалансированный боец.
    - **Волшебник (Wizard):** Обладает магическим щитом, повышающим защиту в бою.
    - **Боец (Fighter):** Использует супер-атаку для нанесения огромного урона.
- **Система боя:** Пошаговые сражения с расчетом урона и защитой.
- **Система прокачки:** Побеждая в боях, покемоны получают опыт, повышают уровень и увеличивают свои характеристики.

---

## Команды бота

| **/go** или `Создать покемона` | Создает нового покемона. Позволяет выбрать класс (Обычный, Волшебник, Боец или случайно). |
| **/attack** или `Атаковать` | Открывает список живых игроков для атаки. Выбери цель и начни бой! |
| **/heal** или `Лечение` | Полностью восстанавливает здоровье твоего покемона. |
| **/info** или `Моя статистика` | Показывает детальную информацию о твоем покемоне (уровень, опыт, характеристики). |
| **/players** или `Игроки` | Выводит список всех активных тренеров и статус их покемонов. |
| **/help** или `Помощь` | Показывает это сообщение с подсказками. |

---

## Технологии и структура

Проект написан на **Python** с использованием следующих библиотек:

- `pyTelegramBotAPI` — для взаимодействия с Telegram Bot API.
- `requests` — для получения данных о покемонах из PokeAPI.
- `datetime` — для управления таймерами.
